新的地界

Posted on 2022-04-06 In 想法

为什么要换新的博客？

起因是今天突然想写点东西了，却发现没有一个可靠的博客地：

  • 提供博客服务的网站：CSDN？条条框框太多，影响使用
  • 可以发布文章的网站：知乎？东西太多太杂
  • OI 评测网站的博客功能：洛谷？已经很少访问，不好管理

最后还是回到 GitHub Pages 这个选项。不过，之前的做法大概是手动使用 typora 生成一些 HTML 文件，然后上传到 GitHub。

因为确实丑的同时又很麻烦，就顺手安装了一个 HEXO 来进行管理。同时，也把一些旧的比较有参考意义的题解给搬到博客上来了。

倒是突然一看之前的题解已经是 2018 年的文字了，这几年的想法是否全部从脑海一闪而过了呢？不得而知。
